Detailed Description

This class can be regarded as a template for low-level position control of the robot.

This class can be regarded as a template for UDP communication client.

This class can be regarded as a template for UDP communication server.

This class used for wireless handle button and joystick detection.

This class can be regarded as a template for high-level movement control of the robot.

This class can be regarded as a template for low-level velocity control of the robot.

This class can be regarded as a template for low-level torque control of the robot.

The class integrates the parameters of user sending and receiving data, three member functions, which are respectively used to receive robot state data, send motor commands and generate motor commands according to preset user commands
